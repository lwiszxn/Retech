# ♻️ Retech - Sistema de Gerenciamento de Descarte Tecnológico

## 📝 Descrição do Projeto
O **Retech** é uma aplicação web desenvolvida como projeto acadêmico para gerenciar o ciclo de vida de equipamentos de TI. O sistema permite o controle rigoroso desde o inventário até o descarte sustentável de hardware, garantindo rastreabilidade e organização.

## ⚙️ Funcionalidades
* **Autenticação e Gestão de Usuários (CRUD):** * Cadastro de novos usuários com validação de CPF.
  * Login seguro com controle de sessão.
  * Recuperação de senha.
  * Perfil do usuário com edição de dados (Update) e exclusão de conta (Delete).
* **Gestão de Inventário:** Listagem e exclusão de equipamentos cadastrados.
* **Módulo de Descarte:** Registro detalhado de itens de hardware (produto, estado, quantidade e motivo do descarte).
* **Controle de Saída:** Envio de itens do inventário para locais de destino específicos.
* **Histórico e Rastreabilidade:** Visualização de itens enviados e opção de retorno ao inventário principal.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando tecnologias front-end puras, sem dependência de frameworks externos ou bancos de dados em nuvem nesta etapa:
* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização moderna utilizando o conceito de *Glassmorphism* e design responsivo.
* **JavaScript (Vanilla):** Lógica de negócios, manipulação do DOM e controle de rotas.
* **LocalStorage & SessionStorage:** Persistência de dados e gerenciamento de sessão do usuário no navegador.
* **Font Awesome:** Biblioteca de ícones.
